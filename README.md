# Pong en Godot  🏓

Un proyecto clásico de **Pong** desarrollado en [Godot Engine 4](https://godotengine.org/). Este juego fue creado como un proyecto de práctica para entender las bases del motor, incluyendo el manejo de nodos, escenas, físicas 2D (colisiones y rebotes), y el sistema de interfaz de usuario (UI).

## ✨ Características del juego

Además de la jugabilidad clásica de Pong, este proyecto incluye algunas funcionalidades extra que expanden el juego base:

* **Menú Principal:** Una pantalla de inicio interactiva antes de comenzar la partida.
* **Sistema de Pausa:** Posibilidad de pausar y reanudar el juego en cualquier momento presionando un botón en la pantalla.
* **Condición de Victoria:** El juego lleva el conteo de los puntos y finaliza automáticamente cuando uno de los dos jugadores alcanza los **5 puntos**.
